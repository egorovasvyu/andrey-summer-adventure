# Как выложить сайт для Андрея

## Что загружать

Самый удобный набор лежит в папке `upload-ready`.

Можно загрузить содержимое этой папки в GitHub или Replit. Главный файл — `index.html`.

## Вариант 1. Replit

1. Создайте новый проект.
2. Выберите HTML/CSS/JS или импорт из GitHub.
3. Загрузите файлы из `upload-ready`.
4. Запустите проект.
5. Replit откроет сайт из `index.html`.

## Вариант 2. GitHub

1. Откройте репозиторий.
2. Загрузите содержимое папки `upload-ready`.
3. Нажмите `Commit changes`.
4. Дальше Replit может импортировать проект из GitHub.

## Важно про данные

Supabase уже подключён через `supabase-config.js`.

Для базы нужна таблица из `supabase-minimal-state.sql`. Она хранит полный снимок приложения: календарь, книги, миссии, награды, родительские настройки и текущий день.
