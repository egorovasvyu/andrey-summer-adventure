# Как выложить сайт для Андрея

## Самый простой путь

Используем:

- GitHub — хранит файлы сайта.
- Vercel — даёт ссылку на сайт.
- Supabase — позже будет хранить реальные данные.

## Шаг 1. Создать GitHub-репозиторий

1. Откройте https://github.com.
2. Войдите в аккаунт или создайте его.
3. Нажмите `+` → `New repository`.
4. Название: `andrey-summer-adventure`.
5. Выберите `Public` или `Private`.
6. Нажмите `Create repository`.

## Шаг 2. Загрузить файлы

Вариант без команд:

1. На странице нового репозитория нажмите `uploading an existing file`.
2. Перетащите туда файлы из этой папки:
   - `index.html`
   - `app.js`
   - `styles.css`
   - папку `assets`
   - `README.md`
   - `vercel.json`
   - `supabase-schema.sql`
3. Нажмите `Commit changes`.

Можно также распаковать файл `andrey-summer-adventure-upload.zip` и загрузить всё из него.

## Шаг 3. Выложить на Vercel

1. Откройте https://vercel.com.
2. Войдите через GitHub.
3. Нажмите `Add New Project`.
4. Выберите репозиторий `andrey-summer-adventure`.
5. Ничего не меняйте в настройках сборки.
6. Нажмите `Deploy`.

Через минуту появится ссылка на сайт.

## Важно про данные

Сейчас сайт сохраняет данные в браузере. Для настоящего семейного использования с разных устройств следующим шагом надо подключить Supabase.

Файл `supabase-schema.sql` уже подготовлен как основа базы данных.
