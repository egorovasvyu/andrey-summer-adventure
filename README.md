# Лето Андрея: 99 ночей в лесу

Кликабельный веб-прототип летнего трекера для Андрея.

## Что внутри

- `index.html` — главная страница для хостинга.
- `app.js` — логика приложения и демо-данные.
- `styles.css` — визуальный стиль.
- `assets/night-forest-camp.png` — фон первого экрана.
- `andrey-summer-adventure.html` — автономная версия одним файлом для открытия с компьютера.
- `supabase-config.js` — подключение к Supabase.
- `supabase-minimal-state.sql` — минимальная таблица для сохранения состояния приложения.
- `upload-ready` — чистая папка с файлами для загрузки на GitHub/Replit.

## Как открыть на компьютере

Откройте `index.html` или `andrey-summer-adventure.html` в браузере.

## Как выложить на Replit или GitHub

Для загрузки используйте файлы из папки `upload-ready` или архив `andrey-summer-adventure-upload.zip`.

Главный файл сайта — `index.html`.

## Где сейчас хранятся данные

Данные сохраняются в Supabase в таблицу `app_state_documents`.

Если Supabase временно недоступен, приложение старается сохранить данные в браузере как запасной вариант.
