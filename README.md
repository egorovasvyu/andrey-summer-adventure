# Лето Андрея: 99 ночей в лесу

Кликабельный веб-прототип летнего трекера для Андрея.

## Что внутри

- `index.html` — главная страница для хостинга.
- `app.js` — логика приложения и демо-данные.
- `styles.css` — визуальный стиль.
- `assets/night-forest-camp.png` — фон первого экрана.
- `andrey-summer-adventure.html` — автономная версия одним файлом для открытия с компьютера.
- `supabase-schema.sql` — черновик таблиц для будущей базы данных.

## Как открыть на компьютере

Откройте `index.html` или `andrey-summer-adventure.html` в браузере.

## Как выложить на Vercel

1. Создайте новый репозиторий на GitHub, например `andrey-summer-adventure`.
2. Загрузите туда файлы из этой папки.
3. Зайдите на https://vercel.com.
4. Нажмите `Add New Project`.
5. Выберите репозиторий `andrey-summer-adventure`.
6. Build command оставьте пустым.
7. Output directory оставьте пустым.
8. Нажмите `Deploy`.

После этого Vercel даст ссылку на сайт.

## Где сейчас хранятся данные

Пока данные сохраняются в браузере через `localStorage`. Это подходит для прототипа и одного устройства.

Чтобы Андрей пользовался с разных устройств, надо подключить Supabase. Для этого можно использовать файл `supabase-schema.sql` как основу базы.
